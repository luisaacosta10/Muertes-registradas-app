# Aplicación Web - Proporción de Muertes Registradas

Esta es una aplicación web interactiva que analiza la proporción de muertes registradas por país a lo largo del tiempo, usando datos de Our World in Data.

## Cómo usar

1. Asegúrate de tener Node.js y npm instalados.
2. Descarga o clona este repositorio.
3. Ejecuta:

    npm install
    npm run dev

4. Abre tu navegador en http://localhost:3000

## Tecnologías usadas

- React
- Recharts
- Leaflet (preparado para mapas futuros)
- Vite

## Dataset

- `share-of-deaths-registered.csv` en `src/data/`
